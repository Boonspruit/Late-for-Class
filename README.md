# Late for Class!

An endless stair-climbing reflex game built with **Phaser 3**. Help the student race up the stairs before the bell rings! Avoid tripping and see how high you can climb.

## Gameplay
- The student automatically runs up the stairs.
- Press **LEFT ARROW** or **RIGHT ARROW** (or tap the left/right sides of the screen) to match the stair direction.
- Survive increasingly difficult patterns, including sudden specific complex jukes!
- Don't step the wrong way or you'll trip and it's Game Over!

## Features
- Dynamic procedural stair generation.
- Responsive design tailored for mobile browsers with touch controls.
- Endless scrolling and scaling difficulty.
- Local high-score tracking using the browser's `localStorage`.
- AI Generated beautiful dynamic 2D Pixel Art and UI using Phaser.

## How to Run Locally
1. Clone the repository.
2. Ensure you have Node.js installed.
3. Run `npm install` to install Phaser 3 and Vite dependencies.
4. Run `npm run dev` to start a local Vite development server.
5. Open the game in your browser at `http://localhost:5173` (or the port specified in your console).
